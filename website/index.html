Projects

Project 1: Interactive U.S. State Income & Demographics Visualization

This project is a dynamic data visualization built with p5.js that displays median household income and demographic distributions for all 50 U.S. states. Users can select a state from a dropdown menu to view an interactive bar chart of median incomes by race and a pie chart showing the racial composition. The visualization automatically resizes to fit the browser window, ensuring a responsive and user-friendly experience.
